# Image to PPT

`image-to-ppt` 是一个用于“从资料到 PPT 可用图片模块”的 Codex skill。它适合把论文、Word、PDF、PPT 截图、仿真图、参考图片和对话内容整理成可直接放入 PPT 的概念图、小卡片、标注图和排版方案。

这个 skill 的目标不是单纯生成一张图片，而是形成一套更适合 PPT 使用的视觉资产：内容精炼、风格统一、比例合适、文字清晰、便于手动或自动放入 PPT。

## 功能

- 从 PDF、Word、截图、图表和对话中提取 PPT 相关信息。
- 将长段技术内容压缩为适合答辩展示的短句。
- 根据已有 PPT 截图或模板提取标题、背景、配色和卡片风格。
- 生成研究方法、仿真结果、算法优势、硬件实现等概念图。
- 将复杂页面拆分为多个可单独插入 PPT 的小图模块。
- 输出高清 PNG，适合直接插入 PowerPoint。
- 在需要时给出图片放置建议，或辅助完成 PPT 页面排版。

## 适用场景

- 根据毕业论文生成“研究方法与过程”页面。
- 在仿真曲线图上添加箭头、注释和算法优势说明。
- 为仿真结果页生成右侧结论卡片。
- 将仿真设置内容整理成少文字的小卡片和表格。
- 根据论文与参考页生成 FPGA/SDR 平台实现方案图。
- 对已有 PPT 图进行裁剪、高清化和锐化处理。

## 推荐流程

1. 提供资料：PDF、Word、PPT 截图、图表或文字说明。
2. 如果需要风格统一，提供当前 PPT 页面截图或模板。
3. 当页面结构不确定时，先生成一张整体概念布局图。
4. 再分模块生成小卡片、横向条、标注图、结构图等独立 PNG。
5. 将生成图片插入 PPT 模板中。
6. 根据实际页面效果继续调整比例、措辞、清晰度和学术表达。

## 示例提示词

```text
使用 $image-to-ppt 阅读我的毕业论文和当前 PPT 截图，生成一页研究结论展示的排版方案，并分别输出右侧结论卡片和仿真图标注图片。
```

## 注意事项

对于中文 PPT 内容，建议优先使用 PIL、SVG 或 HTML 等确定性绘图方式生成文字和图表，避免纯图片生成导致中文模糊、乱码或文字越界。
